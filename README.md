# 生物体科学

## 收服其心之法[[1]](https://en.wikipedia.org/wiki/Seraphim_Falls)[[2]](https://en.wikipedia.org/wiki/Sherlock_(TV_series))[[3]](https://en.wikipedia.org/wiki/Sh%C5%8Dgun_(2024_TV_series))

起后需伏。伏，不达，则更起而伏。

伏后需起。起，不达，则更伏而起。

能起而伏之，能伏而起之，则得之。

苦后需乐。乐，不达，则更苦而乐。

乐后需苦。苦，不达，则更乐而苦。

能苦而乐之，能乐而苦之，则得之。
